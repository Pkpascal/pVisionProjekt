Basic Chips <mat-chip>displays a list of values as individual, keyboard accessible, chips. Official Component Chip input The MatChipInput directive can be used together with a chip-list to streamline the interaction between the two components. This directive adds chip-specific behaviors to the input element within for adding and removing chips. Stacked Chips You can also stack the chips if you want them on top of each other and/or use the (focus) event to run custom code.
